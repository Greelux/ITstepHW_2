HyperText Markup Language (HTML)

HTML — стандартна мова розмітки для документів, призначених для перегляду у веббраузері. Вона описує структуру сторінки за допомогою елементів (тегів).

Cascading Style Sheets (CSS)

CSS — мова стилів, що визначає зовнішній вигляд HTML: кольори, шрифти, відступи, сітки, адаптивність та інше оформлення.

JavaScript (JS)

JavaScript — мова програмування для інтерактивності у вебі: обробка подій, робота з DOM, запити до серверів, анімації тощо.
